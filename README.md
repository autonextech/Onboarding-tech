# Onboarding-tech

A production-grade employee onboarding platform featuring candidate management, interactive module builders, real-time analytics, and mentorship tracking. 

## Tech Stack
- **Frontend**: React 18, Vite, Tailwind CSS v4, Zustand, React Router DOM, Framer Motion, Recharts
- **Backend**: Node.js, Express, Prisma ORM, PostgreSQL, Bcrypt, JWT
- **Design System**: Glassmorphism, Navy Blue & Sky Blue theme

## Setup Instructions

### 1. Database
Ensure PostgreSQL is running locally.

### 2. Backend
```bash
cd backend
npm install
npm run dev
```

### 3. Frontend
```bash
cd frontend
npm install
npm run dev
```
